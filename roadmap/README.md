# Project Roadmap

This project will evolve to simulate a more advanced security infrastructure.

## Phase 1 – Local Lab

Current implementation:

- Ubuntu Server VPN gateway
- WireGuard encrypted tunnel
- UFW firewall hardening
- Fail2Ban brute-force protection

## Phase 2 – Cloud Security Infrastructure

Future improvements will include:

- Cloud deployment of the VPN gateway
- Intrusion detection using Suricata
- Honeypot environment for attack observation
- Centralized log monitoring with Wazuh
- Network segmentation using pfSense firewall

The objective is to simulate a small defensive security architecture.
